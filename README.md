# BST Token (Bridged Stable Token)

## Overview
BST is a TRC20 token deployed on the Tron blockchain network.
Built for research and development purposes to explore 
TRC20 token standards and decentralized finance primitives.

## Token Details
- **Network:** Tron Mainnet
- **Standard:** TRC20
- **Symbol:** USDTb
- **Decimals:** 6
- **Total Supply:** 500,000,000
- **Contract:** TXjv3CKisENooamLFGjyUfmgczADaHvn57

## Features
- Blacklist and whitelist management
- Sell tax mechanism
- Owner controlled trading
- Mintable supply
- Ownership transferable
